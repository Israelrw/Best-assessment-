🚀 Getting Started

1
Create Your Account

Visit the login page and click "Create Account". Fill in your name, username, email and password. Your account is created instantly — no email verification needed.

2
Access the Admin Panel

After signing in, you land on the Admin Panel where you can create and manage your tests. The sidebar shows all your tests; click any to edit it.

3
Free Plan Includes 5 Tests

Your free account lets you create up to 5 tests. Once you reach the limit, you'll see an upgrade prompt. Upgrade anytime from the Plans page.

📝 Creating Tests

1
Click "+ New" in the Sidebar

A new blank test is created instantly. Give it a title, set the duration (minutes) and pass mark (%).

2
Add Questions

Choose from three question types: MCQ (4 options, select the correct one), True/False, or Short Answer (enter accepted answers separated by commas).

3
Add Images to Questions

Each question has an "Add Image" button. Click it to upload a photo, diagram or chart. Supported formats: JPG, PNG, GIF.

4
Use the Equation Editor

For Maths, Physics or Chemistry questions, click the ⚛️ button on any question. Use the toolbar buttons or type LaTeX directly (e.g. \frac{-b \pm \sqrt{b^2-4ac}}{2a}). Click Preview, then Insert.

5
Schedule Your Test

Optionally set a Start Date and End Date. The test will only appear on the student portal within that window. Outside the window, it's hidden from students.

6
Save the Test

Click "💾 Save Test". A shareable link is generated automatically above the questions section.

🔗 Sharing with Students

1
Copy the Quiz Link

After saving a test, the "Share Link" box shows a unique URL. Click "Copy" and share it via WhatsApp, email, or any messaging platform.

2
Students Enter Their Details

When a student opens the link, they see a form asking for their Full Name and Student ID. They must fill both before the exam begins.

3
Exam Auto-Maximizes

The browser automatically requests fullscreen mode when the exam starts. Exiting fullscreen is logged as a cheating violation.

🛡️ Anti-Cheat System

1
What Is Detected

The system detects: switching to another tab, minimizing the browser, losing window focus, exiting fullscreen, and keyboard shortcuts (Ctrl+C, F12, etc.).

2
Warning System

Each violation shows a red warning overlay with a countdown. The timer is paused until the student dismisses the warning and returns to the exam.

3
Automatic Block

When violations reach the maximum (default: 3, configurable per test), the exam is automatically submitted with whatever answers were given, and the student is blocked.

4
Configure Per Test

Each test has its own anti-cheat settings: enable/disable it, set max violations, choose whether to shuffle questions and options.

📊 Viewing Results

1
Access the Results Page

Click "Results" in the top navigation. You'll see all student submissions for tests you own, sorted by most recent.

2
Filter and Search

Use the search box to find a specific student by name or ID. Filter by test or by pass/fail status.

3
Export to Excel

Click "📊 Export Excel" to download all filtered results as a CSV file that opens perfectly in Microsoft Excel or Google Sheets. Each row contains student info, score, percentage, status and violation count.

4
Student View of Results

After submitting, students see their score, pass/fail status and time taken. Correct answers are shown only if you enabled "Show Correct Answers" in the test settings.

⚛️ Equation Editor

1
For Maths Questions

Examples: \frac{a}{b} for fractions, \sqrt{x} for square roots, x^{2} for powers.

2
For Chemistry

Use \ce{H2O} for chemical formulas, \ce{->} for reaction arrows.

3
For Physics

Use \vec{F} for vectors, \int_{a}^{b} for integrals, \Delta for delta.
